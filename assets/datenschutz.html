Datenschutzerklärung

Diese App ist eine Progressive Web App (PWA), die auf Ihrem Gerät ausgeführt wird.

Datenspeicherung

Alle Daten, die Sie in dieser App eingeben (z. B. in Formularen), werden ausschließlich lokal auf Ihrem Gerät im Speicher Ihres Webbrowsers verarbeitet. Es findet keine Übertragung dieser Daten an einen externen Server statt.

NFC-Funktion

Die Web NFC API wird verwendet, um direkt mit NFC-Tags zu kommunizieren. Die gelesenen und geschriebenen Daten verlassen Ihr Gerät nicht, es sei denn, Sie speichern sie explizit als JSON-Datei.

Caching

Die App-Dateien werden lokal auf Ihrem Gerät zwischengespeichert (gecacht), um eine Offline-Nutzung zu ermöglichen. Dies dient ausschließlich der Funktionalität und Performance.
